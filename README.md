# Key Stack Overlay Project

## ✨ Motivation

When people see someone studying or working in Computer Science,  
it feels kind of cool when the keyboard typing is shown on the screen. 😂

So I thought:

> “What if I make a small program that shows my typing on the screen?”

Since Codex is also available now,  
I wanted to use Codex and quickly build this project.

---

## 🧠 Project Idea

These days, **vibe coding** is becoming very popular.

But even when using tools like Codex or Claude Code,  
I think the programmer still needs to understand:

- what libraries are being used
- how the program works
- how the whole flow is structured

So before using Codex,  
I first planned the basic flow of the program.

---

## ⚙️ Main Features

The program should work like this:

- Show the letters typed by the user
- Display them on the bottom-left side of the screen
- Run in the background
- Support ON / OFF toggle
- Show up to 3 lines
- Each text disappears after 5 seconds
- Make the typing effect look cool and clean

---

## 🚀 Why This Project Is Fun

Tools like Codex and Claude Code make coding much easier now.

Honestly, vibe coding feels really convenient.

But I still think it is important to understand the project structure first.  
That way, AI can help faster and better.
